# Changelog

## 1.0.0

Initial release.

- Added selection, current paragraph, quick panel, and insert-below translation workflows.
- Added AI and official machine translation provider configuration.
- Added prompt presets for AI translation.
- Added immersive bilingual reading and side-by-side Markdown document translation.
- Added local translation cache, request queue limits, and runtime dashboard metrics.
- Added image translation actions for embedded Markdown images.
- Added TTS settings and floating result panel speech playback.
- Added interface languages: Simplified Chinese, English, German, and Japanese.
- Added privacy disclosure in the settings About tab and README.
